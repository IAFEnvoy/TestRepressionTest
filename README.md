# 测试压抑度量表

简易静态页面，包含题目表格与结果显示。页面以浅色为主，答题方式为每题选择 1～5 分（1=非常不符合，5=非常符合）。

如何运行：

- 直接在浏览器中打开：打开文件管理器，双击打开 `index.html` 即可（适合现代浏览器）。
- 或用本地静态服务器（推荐以避免某些浏览器的本地脚本限制）：

```bash
# 使用 Python 3 的内建服务器（在项目根目录运行）
python -m http.server 8000

# 或使用 npm 的 http-server（需先安装：npm i -g http-server）
http-server -c-1
```

说明：本测试仅供娱乐，不作为任何心理诊断或评估依据。
